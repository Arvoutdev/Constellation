Setup

All repeating logic is now placed via /setblock instead of printed as text to hand-place. Each floor gets its own row of repeating command blocks laid out along +X, starting at the "Command block area" point above, one row per floor (2 blocks apart in Z so rows don't touch). They're placed as type=repeating but auto:0b - meaning NOT always active. You still need to power each row with redstone yourself (a lever + block, or a permanent redstone block on the first block block of each row) before anything will run.
